# WEB3 - Project - API

## Configuration
- PHP: supérieur ou égale à 8.0
- Composer: 2.0.12
- MYSQL: 5.7.31

## Installation

- Dupliquer le fichier **.env** en **.env.local** qui ne sera pas versionné.
- Renseigné les différentes varianble d'environnement

Si vous êtes en développement exécutez cette commande:
```
make init_local
```
